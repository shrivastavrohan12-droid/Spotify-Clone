# Spotify Clone

A frontend clone of Spotify's UI built to practice HTML, CSS, and Vanilla JavaScript with responsive layouts and DOM manipulation.

## 🛠️ Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (ES6+)

```bash

# No installation required! Just open index.html in your browser.
```
